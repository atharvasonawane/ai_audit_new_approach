# Frontend Rules

Use:
- Vue 3
- TypeScript
- Tailwind
- Composition API

Design Rules:
- Minimal UI
- Large whitespace
- Neutral palette
- Subtle borders
- Rounded-xl cards

UX Rules:
- Prioritize readability
- Avoid clutter
- Use compact data tables
- Charts only when meaningful

Code Rules:
- Reusable components
- Clean folder structure
- No giant files
- Use composables

Animation:
- Subtle only
- No flashy motion
